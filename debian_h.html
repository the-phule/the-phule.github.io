HTML Build for git based static website.

Idea:
Allow a simple command line creation of static html for github hosted websites.

From the head.html, base.html, foot.html:


<!DOCTYPE html>
<html>
  <head>
    <title>Random Musing</title>
  </head>
  <body>
    <header>
      <h3>
	<!-- Insert title here: -->
	title
      </h3>
    </header>
<!-- WARNING DO NOT PUT ANYTHING BELOW THIS COMMENT -->
~/base/head.html

cat the outputs, in order, to the file.


$ cat head.html >> test.html
$ cat base.html >> test.html 
$ cat foot.html >> test.html

Check output file in both a text editor and browser, at this point if something is incorrect delete the output file. Edit the original base file, and re-cat the file. 


$ emacs test.html &
$ iceweasel test.html &
